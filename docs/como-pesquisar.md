# Como Pesquisar

A busca do portal é fornecida pelo sistema de indexação do Material for MkDocs, baseado em análise lexical e correspondência por relevância.  
Os resultados consideram título, corpo do texto, termos derivados e combinações aproximadas.

## Pesquisas simples
Digite palavras-chave como *“servidor efetivo”*, *“tributos”* ou *“contratos”*.  
A ferramenta identifica variações morfológicas (por exemplo, “servidores”, “serviço”, “servir”).

## Aspas para busca exata
Use aspas para localizar expressões completas, como: "estágio probatório"

## Filtros naturais
Combinações de termos funcionam como filtros contextuais.  
Exemplo: **educação conselho** retornará normas que tratem simultaneamente dos dois assuntos.

## Limitações
A busca não interpreta conceitos jurídicos complexos; trabalha apenas com texto literal.  
Sempre confirme a leitura integral da norma para interpretação adequada.

Para aprofundar o método, consulte a documentação do Material for MkDocs: https://squidfunk.github.io/mkdocs-material/setup/setting-up-site-search/
